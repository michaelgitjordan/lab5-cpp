# Класс LabWork: Работа с бинарными файлами
## Назначение
Класс реализует функционал для работы с бинарными файлами, содержащими:

1.Числовые данные (целые числа) для выполнения задания 1

2.Структурированные данные (объекты Toy) для выполнения задания 3

## Основные возможности
- Задание 1: Работа с числовыми файлами
1. fillFileWithRandomNumbers() - создает бинарный файл и заполняет его случайными целыми числами

2. findDifferenceFirstAndMax() - находит разность между первым и максимальным элементами файла

- Задание 3: Работа со структурами игрушек
1. createToy() - интерактивное создание объекта Toy с проверкой вводимых данных

2. createToyFile() - создание нового бинарного файла с игрушками

3. addToyToFile() - добавление игрушки в существующий бинарный файл

4. calculateDollCostForAge() - вычисление общей стоимости кукол для заданного возраста

## Вспомогательные методы
1. displayFileContents() - отображение содержимого числового бинарного файла

2. displayToyFileContents() - отображение содержимого файла с игрушками в табличном формате

3. printToy() - форматированный вывод информации об игрушке

## Особенности реализации
- Все методы являются статическими - не требуют создания экземпляра класса

- Работа с файлами исключительно в бинарном режиме (std::ios::binary)

- Автоматическая проверка корректности операций ввода/вывода

- Защита от переполнения буфера при копировании строковых данных

## Структура данных Toy
Класс использует структуру Toy для хранения информации об игрушках:

1. Название (массив char[50])

2. Стоимость (double)

3. Минимальный возраст (uint8_t)

4. Максимальный возраст (uint8_t)

5. Флаг "является куклой" (bool)
